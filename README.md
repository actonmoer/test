# vue-book
## 项目说明
> 
1.本项目使用vue-cli创建，没有使用ui框架，自己封装了一些小组件都放在components文件夹里面    
2.本项目使用vue-awesome-swiper做为轮播的插件    
3.本项目使用axios插件做为http请求插件，api文件放在public/api    
4.CountDown.js 倒计时类     
5.Resources.js 图片资源 
>   

## Project setup
```
yarn install
```
